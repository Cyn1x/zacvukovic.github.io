About Me

UniChess Chess Engine

Real-time chess engine simulation that detects valid moves for an active piece before the player has requested a position to move to. The board state is managed using Forsyth-Edwards Notation, with the added ability to specify a board state to play with. The README in the source code repository explains the functionality in more detail.


Written in TypeScript and rendered with React + HTML5 Canvas.

QuadtreeFX

Quadtree that isolates areas on the cartesian plane for collision detection, so each individual entity is not wasting system resources by checking if it is colliding with every entity in memory. Instead, each entity is placed into a leaf node depending on which cardinal direction the entity is in, and then the collision detection is computed with other entities in that leaf node.


Written in Java and rendered with JavaFX.
